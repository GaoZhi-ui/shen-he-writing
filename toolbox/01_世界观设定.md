# 世界观设定

> 赛马娘世界·写作参考精简版

---

## 世界基调

现代日本 + 赛马娘存在。科技水平与现实日本一致。
没有真实的马。赛马娘是代替马的类人种族。

## 核心机构

| 机构 | 职能 |
|------|------|
| URA | 赛事运营、学园管理、行业监管 |
| 中央特雷森学园 | 东京府中市，全国最顶尖的赛马娘培养机构 |
| 地方特雷森 | 全国15所分校 |

## 比赛体系

未胜利战 → 条件赛(500万下/1000万下/1600万下) → OP → GIII → GII → GI

**经典三冠（牡马路线）**：皋月赏(2000m) → 日本德比(2400m) → 菊花赏(3000m)

**经典三冠（牝马路线）**：樱花赏(1600m) → 橡树大赛(2400m) → 秋华赏(2000m)

## 马娘能力

**脚质**：逃(领放) · 先(前列) · 差(差行) · 追(追込)

**距离适性**：短距离(1000-1400m) · 英里(1400-1600m) · 中距离(1800-2400m) · 长距离(2500m+)

**场地**：芝(草地) · ダート(泥地)

## 世界观铁律

- 赛马娘世界 = 现代日本 + 马娘存在
- 比赛中马娘可达60km/h+，日常与常人无异
- 不许出现泰拉/方舟等无关设定
- 日本现实地名（东京竞马场/府中/京都/阪神等）正常使用
