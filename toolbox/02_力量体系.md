# 力量体系

> 赛马娘的身体能力体系 · 写作参考

---

## 五维属性

| 属性 | 作用 | 沈禾初始 | 成长方向 |
|------|------|---------|---------|
| 速度 | 最高速度上限（终盘决胜属性） | B+ | 目标A |
| 耐力 | 维持速度的能力（体力门槛） | B | 目标B+ |
| 力量 | 加速度·爬坡·变道 | B+ | 目标A- |
| 根性 | 终盘耐力节约·最低速度 | A- | 目标A |
| 智力 | 技能触发·节奏判断·路径选择 | A | 目标A |

## 跑法类型

### 战术跑法（策略选择）
| 跑法 | 特点 | 适合 |
|:----:|:----:|:----:|
| 逃 | 从头领放到尾 | 耐力型、自信型 |
| 先 | 跟在前列，伺机而动 | 均衡型 |
| **差** | 中团待机，末脚决胜 | **沈禾·爆发型** |
| 追 | 最后方等待，直线超车 | 末脚特化型 |

### 物理跑法（身体类型）
| 类型 | 特点 | 适合距离 |
|:----:|:----|:--------:|
| Stride跑法 | 大步幅，步频较低 | 长距离 |
| Pitch跑法 | 小步幅，快步频 | 短距离 |
| 中间型 | 介于两者之间 | 中距离 |

## 比赛节奏

```
出闸 → 序盘(0~1/6) → 中盘(1/6~1/2) → 终盘(1/2~终点)
        抢位       节奏保持       胜负决断
```

沈禾的差行展开：出闸·中团待机 → 中盘·控制节奏 → 最后弯道·外移 → 直线上·末脚爆发

## 体力公式（参考）

体力 = 赛道长度 + 0.8 × 跑法系数 × 耐力属性

体力耗尽 = 失速（减速到最低速度匀速前进）
