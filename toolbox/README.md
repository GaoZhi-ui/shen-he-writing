# 沈禾·写作工具箱

## 目录

| 编号 | 文件 | 用途 |
|:----:|:----|:------|
| 00 | 故事核心 | 一句话梗概、主题、情感核、结局定位 |
| 01 | 世界观设定 | 赛马娘世界的基础设定精简版 |
| 02 | 力量体系 | 五维属性、跑法类型、比赛节奏、术语 |
| 03 | 角色管理 | 主要角色概要、待定角色 |
| 04 | 伏笔管理 | 已埋/待埋伏笔追踪、回收计划 |
| 05 | 时间线 | 故事内部时间轴 |
| 06 | 人物关系图 | 角色关系网络 |
| 07 | 情节脉络 | 全篇关键节点、情绪曲线、转折点 |

## 使用方式

- 每写一章前查看04_伏笔管理.md（有没有要推进/回收的伏笔）
- 每写完一卷更新05_时间线.md
- 新角色登场时更新03_角色管理.md + 06_人物关系图.md
- 00_故事核心.md 不动——写偏了的时候回来读一遍
